Audible sign in required

LibraForge uses Audible to look up book metadata. You need to connect an Audible account before running the Metadata Forge or M4B Tool.

Recommendation: use a dedicated empty Audible account for metadata lookups rather than your primary account. No purchases, downloads, or library access are used — only the product catalogue search API.

Encrypted auth files are not supported. The generated auth file is stored unencrypted at /auth/audible-metadata.json inside the container (mounted from the host path you configured).
